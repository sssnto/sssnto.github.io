Github Pages设置

什么是Github Pages

GitHub Pages 本用于介绍托管在GitHub的项目，不过，由于他的空间免费稳定，用来做搭建一个博客再好不过了。

每个帐号只能有一个仓库来存放个人主页，而且仓库的名字必须是username/username.github.io，这是特殊的命名约定。你可以通过http://username.github.io 来访问你的个人主页。

Travis CI集成

1.修改git地址

进入.git， 打开config

将url修改成你自己的github地址

2. 修改.travis.xml

修改成自己的用户名和邮箱

3.创建github token

注意：需要复制该token稍后使用，token值只能看到一次。

4. 创建Travis任务

将Github的token添加到目标位置

5. 上传github

将博客push到github即可
